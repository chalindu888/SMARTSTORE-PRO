# 🛍 SmartStore Pro

A modern, data-driven Single Page Application (SPA) built using Vue 3, TypeScript, and Tailwind CSS.

---

## 🚀 Project Overview

SmartStore Pro is a responsive e-commerce frontend application that demonstrates:

- API integration
- Component-based architecture
- State management using Pinia
- Search & filtering
- Authentication simulation
- Dark mode
- Clean Git workflow

This project follows clean architecture principles and modern frontend best practices.

---

## 🛠 Tech Stack

- Vue 3 (Composition API)
- TypeScript
- Tailwind CSS
- Pinia (State Management)
- Vue Router
- Vite

---

## 📦 Features

### ✅ Core Features
- Product listing from API
- Product detail view
- Responsive UI
- Search functionality
- Category filtering

### 🛒 Cart (Bonus)
- Add to cart
- Remove from cart
- Quantity management
- Total price calculation
- Persistent cart (localStorage)

### 🌙 Dark Mode (Bonus)
- Toggle light/dark mode
- Theme saved in localStorage
- Fully styled dark UI

### 🔐 Authentication Simulation (Bonus)
- Login page
- Protected routes
- Route guards
- Persistent login state
